This is a Dungeons and Dragons Generation Program Tool (D&DGPT)

The purpose of this program is to streamline the creation of a level 1 character
for the game Dungeons & Dragons, using information from the 5th edition Player's Handbook.

To use this program, all you need to do is run the program and follow the prompts.

The program will display the bare bones of a new character sheet as calculated by the program.
